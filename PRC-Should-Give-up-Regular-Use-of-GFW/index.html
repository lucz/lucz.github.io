放棄大面積 GFW 是維穩的有效手段


配樂：沒有共產黨就沒有新中國

h2. 前言

GFW (The Great Firewall of China) 始自國外對中國金盾工程的戲稱 1 ，也就是中國用來封鎖境外網站和網站敏感詞的手段。將此名詞意動使用時，則表示封鎖之行為。 GFW 出現初期，封鎖的網站並不多，因而影響不大，網民對他的存在也沒有很明顯的意識。但隨著 GFW 的一步步發展，他已為大部分網民所知。
GFW 的存在，在領導人看來，是維穩的有效手段之一。但經我老人家研究發現，現在的 GFW 不但對維穩功效不大，反而有害維穩。以下將分四大點進行論述。

停用 GFW 不會造成動蕩

中國網民在交流上是與世界非常脫節的：多數中國網民不但未掌握一門可以正常閱讀的外語，而且繁體中文都覺得讀來吃力。因此，多數中國網民在一般情況下不願意閱讀簡體中文以外的任何文字，繁體中文還勉強可讀，其他語言是根本不讀的。
從語言角度我們就可以論證，開放 GFW 後中國網民閱讀文章範圍並不會明顯增加，而增加的文章中，對中國維穩似乎有害的，絕大多數來自輪子的網站。輪子在中國名聲已臭，一般人視之為邪教，他們的媒體在中國不具有公信力，故其文章接受度不高，不可能大面積破壞中國國家秩序。
何況，目前封禁的網站有很多屬於錯殺之列。譬如 SugarSync 和 Dropbox ，只是網盤而已，一般用以備份個人資料，幾乎沒有人會在上面發佈反動信息，加上速度不快，即使尚未封禁之時，中國用戶也並不算多。而形如 Youtube 之類的視頻網站，雖然可能包含八九的錄像，但八九這件事在中國影響雖大，其精神卻已斬斷，很多人對之頗為冷淡，甚至反感。因為經過多年的教化，多數人都希望中國穩定，自己好安穩賺錢。而 Twitter 和 Facebook 文字太短，鬥嘴可以，煽動無功。

長期封鎖會導致 GFW 作用弱化

目前，要翻牆並非難事。比如我老人家，開一個 MyEnTunnel ，再設置一下瀏覽器就搞定了。又如輪子的自由門，更簡單，自動配置，就是速度差一點。
如今網絡封鎖越來越密集，翻牆的重要性越來越大，則翻牆的人數越來越多，翻牆的方法越來越普及。除非有一日中國啟用 IP 地址白名單制度，不然這些翻牆的方法總是可以用的。一旦有一日，某個境外大流量網站大量發佈煽動信息，中國政府反而措手不及。然而，白名單制度是不可能在短期內實現的，因為應該通過的外國網站實在太多了，而你又不可能要求別人傻乎乎的上來跟你拍照、提交身份證複印件、支付備案費用等等。比如遍佈世界各地大大小小的學校網站，你總不能因為沒備案就封學校網站吧？還有各種慈善、公益機構的網站，封了不是找罵麼？白名單數據庫的建立是需要人工檢閱的，檢閱的文字還是包括但不限於英文、法文、德文、西班牙文、葡萄牙文、阿拉伯文等各種大小語種，就幾年功夫，你看得完嗎？
事實上，美帝就有一個 CNCI ，斥資 300 億美元打造。但是人家是隱而少發的，平時一點事都沒有，哪天說幹掉就幹掉，比如維基解密就死得很難看。

新時代 GFW 功能將出現缺失

在 IPv4 時代，一個美國 IP 的價格大約是 1-2 美元每月，中國的 IP 會貴一些。一般 IDC 是不會隨便給你換 IP 的，所以一個 IP 被封就意味著每月損失 1-2 美元，封幾個之後，小站長一定不玩了的。這是 GFW 的第一狠招，但也會出現漏網情況 2 ，只不過 IPv4 時代大家 IP 少，封不起。然而， IPv6 號稱要給世界上每一粒沙子都編上號，人家組合夠多。他的特點就是極其便宜，便宜到 15 美元一年的 BuyVM 在給了一個 IPv4 的情況下附送 16 個 IPv6 ，而 HE.net 則是免費註冊就送 IPv6 。
另外一招是關鍵詞過濾。現在 ssl 也越來越便宜了，這是有加密功能的，也就是說， GFW 慣用的關鍵詞封鎖法的效果會受到。雖然目前 GFW 重點污染了部分 https ，比如 Google 的，可是對於二線網站，一個一個污染也是很鬧心的。
最後一招 DNS 劫持太土了，直接忽略。
另外域名也開始白菜價了……

停用 GFW 可凝聚向心力

停用了 GFW ，網民就覺得中國更進一步開放了，鉗制少了，對政府的反感也會減少很多。中國網民也會發現，國外也沒什麼好，問題也是那麼的多，國外政府也存在很多問題，而且這些問題越大，越難得到解決。
根據我老人家學過的馬克思主義哲學，矛盾具有普遍性，矛盾無處不在，不時不有；階級矛盾不可調和；矛盾有主要矛盾和次要矛盾，主要矛盾和次要矛盾都是由於事物發展不平衡引起的，並在一定條件下相互轉化 3 。維穩的最佳手段並非壓抑矛盾，因為矛盾的普遍性告訴我們矛盾是不可能壓住的。對付當前國內矛盾的最佳手段應該是轉移矛盾重心，也就是把主要矛盾轉化為中國人民與外國人民的矛盾，而非中國人民與中國政府之間的矛盾。而主次矛盾的轉化方法就是解除網絡封鎖。如果打開 GFW ，對政治感興趣的中國人很可能試探著去上台灣的 BBS ，被台灣深綠民眾罵中國豬，然後回罵。於是我們就可以看到一群中國人一邊罵台灣一邊誇中國政府，再被圍攻、被禁言、封 ID 、封 IP 、封 IP 段，憤怒的他們紛紛到各種中國人的論壇轉帖、貼圖……在此一過程中，中國人對自我與他者的概念將得到強化，從而增加對中國的認同感，堅固國家構成的基礎。這正如抗戰中的國民政府，不管貪污腐化如何，沒人會罵政府，因為主要矛盾是中日矛盾，人們還指望著政府把小日本收拾了。
最後，網絡封鎖是政府不自信的表現，封鎖會讓中國人對外國、對民主格外嚮往。如今政府公信力降至建國以來最低，有沒有 GFW 的一份功勞，我們不得而知。但既然政府尚且不自信，又如何教人相信？解開封鎖，則能表現出一份自信，讓中國人對政府的信心流失得慢一些。
